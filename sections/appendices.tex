\subsection*{Appendix A: Index of the Rocq Formalisation}

This appendix indexes every Rocq definition, function, record and theorem that
appears in a code listing of this thesis. The entries are grouped by topic, following
the order in which they are developed in
Section~\ref{sec:methodology}. The source file of each group is named in its
heading. The full development is available in the
repository~\cite{OurRocqRepo}.
